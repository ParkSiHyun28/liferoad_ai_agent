# My LifeRoad 시연 — 내 노트북에서 돌리기

압축을 풀면 폴더 하나가 나옵니다. 그 폴더 안에서 아래대로 하세요.

---

## 준비물 (Python 3.12)

**그냥 실행 파일부터 더블클릭하세요.** 파이썬이 없으면 커맨드가 알아서 처리합니다.

- 파이썬이 없을 때:
  - **맥** → 파이썬을 자동으로 깔아줍니다.
    - Homebrew가 있으면 바로 설치.
    - Homebrew도 없으면 "Homebrew까지 자동으로 깔까요? y/n" 물어봅니다. `y` 누르면 Homebrew부터 파이썬까지 자동. **중간에 맥 로그인 비밀번호를 한 번 입력**해야 합니다(화면에 안 보임, 입력 후 Enter).
  - **윈도우에 winget이 있으면**(보통 윈10 최신/윈11) → "설치할까요? y/n". `y` 누르면 자동 설치 후, **창을 닫고 다시 한 번 더블클릭**하면 됩니다.
  - **자동 설치가 안 되는 경우** → 설치 페이지가 자동으로 열립니다. 깔고 다시 실행하세요.
- 직접 설치 주소: https://www.python.org/downloads/
  - **윈도우**는 설치 화면에서 **"Add Python to PATH"** 체크를 꼭 켜세요.

---

## 실행 방법

### 맥 (Mac)
1. `시연_시작_맥.command` **더블클릭**
2. 처음 열면 "확인되지 않은 개발자" 경고가 뜰 수 있습니다.
   → 파일에 **마우스 오른쪽 클릭 → 열기 → 열기** 누르면 실행됩니다. (한 번만)
3. 터미널 창이 뜨고 잠시 후 브라우저가 자동으로 열립니다.

### 윈도우 (Windows)
1. `시연_시작_윈도우.bat` **더블클릭**
2. "Windows의 PC 보호" 경고가 뜨면 → **추가 정보 → 실행** 누르세요.
3. 검은 창이 뜨고 잠시 후 브라우저가 자동으로 열립니다.

---

## 처음 한 번은 1~3분 걸립니다

첫 실행 때만 필요한 프로그램을 자동으로 내려받습니다. (인터넷 필요)
두 번째부터는 바로 켜집니다.

브라우저가 자동으로 안 열리면 직접 여세요:
**http://localhost:8000/**

## 잘 안 될 때

| 증상 | 해결 |
|---|---|
| "파이썬이 없다" | 위 주소에서 Python 3.12 설치 후 다시 실행 |
| 패키지 설치 실패 | 인터넷 연결 확인 후 다시 실행 |
| 채팅이 답을 안 함 | `.env` 파일이 폴더 안에 있는지 확인 (API 키가 들어있음) |
| 브라우저가 안 열림 | 직접 http://localhost:8000/ 접속 |
